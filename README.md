# NovaChatProject_Assignment
## Aims / Description
The primary goal of this lab is to explore and demonstrate effective use of GitHub tools for project management and collaborative development while implementing a live messaging feature in an established Application. This functionality is a high priority request from a long-standing client, emphasizing the need for a rapid, quality-assured development process. This lab will detail the use of GitHub Issues, Boards, Pull Requests (PRs), and branching strategies to manage the project efficiently, especially with upcoming team members onboarding in the following weeks.
Objectives:
1.	To track and manage all tasks associated with the live messaging function using GitHub Issues.
2.	To set up GitHub Project Boards for task visualization and progress monitoring.
3.	To manage code integration and collaborative work through Pull Requests.
4.	To utilize effective branching strategies, balancing fast feature development with code stability.
